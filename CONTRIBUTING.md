# 🍞 Contributing to [Let's Do] Bakery 🍞

Thanks for stopping by. I’m happy you’re here to help the mod grow and improve. Pull Requests (PRs) are welcome. Let’s keep everything clean, consistent, and stable.

## How You Can Help

- **Pull Requests (PRs)**: Feel free to submit PRs. Check the existing style before submitting to keep things consistent.
- **Feature Requests**: Got an idea? Open a suggestion and describe what you’d like to see.
- **Bug Reports**: Found a bug? Please write a clear report with steps to reproduce it.

## Guidelines

- **Clean Code**: Keep the code clean and organized. Avoid unused code and unrelated changes.
- **Commit Messages**: Keep commit messages clear and concise. Using [Conventional Commits](https://www.conventionalcommits.org) is recommended.

## Code Style

- Follow the existing code style.
- Make sure the project builds without issues before submitting.

## Testing

- Test your changes thoroughly.
- Add tests if appropriate, or describe what you changed and how it behaves.
